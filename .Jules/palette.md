## 2025-05-08 - Standardized Form Accessibility Pattern
**Learning:** Legacy AngularJS forms often lack semantic labels and keyboard submission support. Using 'sr-only' labels maintains visual design while helping screen readers, and 'ng-submit' on the form with a 'submit' button type enables the 'Enter' key to submit, which is a significant UX win.
**Action:** Always apply 'sr-only' labels and 'ng-submit' patterns to forms encountered in this repository.
