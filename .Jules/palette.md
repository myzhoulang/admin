## 2026-05-30 - Standardized Form UX and Accessibility
**Learning:** Standardizing forms with `.sr-only` labels, `ng-submit`, and the AngularJS "dot rule" (`$scope.f`) improves both accessibility and code maintainability across the app. Verification tools (like Playwright) and their artifacts should not be committed to the production repository.
**Action:** Apply the `sr-only` label and `ng-submit` pattern to all forms. Ensure temporary test dependencies and files are removed before submission.
