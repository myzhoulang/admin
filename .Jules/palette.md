## 2025-05-14 - Refactoring 404 page for better UX
**Learning:** Legacy templates often contain typos and non-functional elements (like a search form with no action). Adding accessibility labels (sr-only) and functional interactions (ng-submit) provides immediate value with minimal code changes.
**Action:** When encountering error pages or common utility pages, check for basic usability: functional forms, clear messaging without typos, and screen reader accessibility.
